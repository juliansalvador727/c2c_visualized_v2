## c2c

Interactive country-to-country shortest-path visualization built with React, TypeScript, and MapLibre.

C2C Visualized is a web application that helps users explore the shortest path between any two countries, highlight visited countries, and interact with global geospatial data through a smooth, intuitive UI.

## stack

nextjs (typescript)
react-map-gl maplibre
geojson (country data)

## getting started

1. clone the repo
   `git clone https://github.com/[name]/c2c-visualized-v2.git
cd c2c-visualized-v2`

2. install dependencies
   `npm install`

3. start the development server
   `npm run dev`

4. open localhost in browser
   `http://localhost:5173`
